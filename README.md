# Wiimote4J

## What's this
Wiimote4J is an unofficial Java library for the Wiimote.

## Fully Supported Devices
### PC
- Windows
- Mac OS X
- Linux

### Controller
- WBC(RVL-021)

## Partially Supported Devices
### Controller
- WCNT(RVL-003)(Button only)
- WNC(RVL-004)(Button only)

## Not Supported Devices
### Controller
- WCC(RVL-005)
- WCP(RVL-005(-02))
- WMP(RVL-026)

## Dependencies
```
JNA
 |
 ---Pure Java HID-API
  |
  ---Wiimote4Java
```
