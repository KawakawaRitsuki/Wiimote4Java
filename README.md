# Wiimote4Java

## What's this
Wiimote4JavaはWiimoteをJavaで使うためのライブラリです。

## Full Supported Devices
### PC
- Windows
- Mac OS X
- Linux

### Controller
- WBC(RVL-021)

## Partially Supported Devices
### Controller
- WCNT(RVL-003)(Button only)
- WNC(RVL-004)(Button only)

## Not Supported Devices
### Controller
- WCC(RVL-005)
- WCP(RVL-005(-02))
- WMP(RVL-026)

## Dependencies
```
Wiimote4Java
 |
 ---Pure Java HID-API
  |
  ---JNA
```
